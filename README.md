# Campus Emergency & Incident Intelligence System (CEIIS)

##  Project Description
The **Campus Emergency & Incident Intelligence System (CEIIS)** is a web-based application designed to provide a structured and secure platform for reporting, managing, and analyzing campus-related incidents. The system allows users to submit incident reports, automatically evaluates the risk level of each incident, and enables administrators to track and resolve incidents efficiently.

This project was developed strictly following the **Software Development Life Cycle (SDLC)**. All system names and nomenclatures used during the design phase are consistently reflected in the implementation.

---

##  Project Objectives
- To enable structured reporting of campus incidents
- To automatically assess and classify incident risk levels
- To support administrators in monitoring and resolving incidents
- To improve campus safety through data-driven decision-making
